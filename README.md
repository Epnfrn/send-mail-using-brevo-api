Programa de CLI escrito en JS que envía correo electrónico utilizando API de Brevo.

Ejemplo de uso: 
`node path/send-mail-using-brevo-api/script.js --to "John D. <jhon@domain.com>" "Doe J. <doe@domain.com>" --subject "Asunto del correo" --content "Contenido textual de correo electrónico. Este es el contenido del correo."`

Requisitos:
- Archivo .env (BREVO_API_KEY=..., BREVO_MAIL=...)
